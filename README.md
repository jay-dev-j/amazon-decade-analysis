# Amazon Decade Analysis Project

## 📊 Overview
This project analyzes Amazon sales data from 2015–2025 using Python, SQL, and Streamlit.

## 🚀 Features
- Customer Growth Analysis
- Revenue Trends
- Market Penetration by State
- Product Inventory Insights

## 🛠 Tech Stack
- Python
- Pandas
- Plotly
- Streamlit
- SQL
